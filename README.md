# 🌐 Personal Portfolio Landing Page

This is my custom solution to the Landing Page project from The Odin Project Foundations curriculum. I built this as a mock-up personal website to showcase my journey as an aspiring web developer, focusing heavily on clean layout structures and interactive elements.

## ✨ Key Features

- **Personal Portfolio Theme:** Customized with sections tailored to introducing myself and my work.
- **Smooth Navigation:** Fully functional header navigation bar with internal anchor links linking directly to the Projects, Info, and Contact sections.
- **Card UI Enhancements:** Styled information cards with custom CSS box shadows to create a modern, elevated look.
- **Call to Action:** An integrated contact section directing users to check out my GitHub profile.

## 🛠️ Built With

- Semantic HTML5
- Custom CSS3 (Flexbox architecture)
- Git & GitHub Pages (for hosting and deployment)

## 🧠 What I Learned & Technical Wins

### Flexbox Alignment & Spacing
While building the multi-column layout for the information section, I got comfortable using Flexbox to space out cards evenly while keeping the subtext aligned neatly underneath the placeholder illustrations.

### Interactive Elements & Polish
- Learned how to implement CSS `:hover` states on buttons to make the interface feel more responsive and dynamic to the user.
- Practiced clean code structure by separating the layout section by section in the stylesheet, matching the flow from the header down to the footer.

## 🔮 Future Improvements

1. **Responsive Media Queries:** Add breakpoints to ensure the layout shifts smoothly from desktop views to mobile screens.
